# 🎤 灵语 (LingYu) — 智能语音输入法

> 语音输入法实训项目 — 颖之妙想队

**灵语**是一款基于本地 AI 的智能语音输入法桌面应用。支持离线语音识别、场景化文本润色、情绪分析与 Emoji 推荐，采用 Electron + Python 混合架构，在 Windows 桌面提供流畅的语音转文字体验。

---

## 功能特性

### 🎯 核心功能
- **离线语音识别** — 基于 FunASR Paraformer 模型，中文识别精准，无需联网
- **场景智能润色** — 办公/聊天/创作三种场景智能优化文案
- **情绪分析 + Emoji 推荐** — 自动识别文本情绪，推荐匹配的表情符号
- **用户词库管理** — 自定义热词、行业词库，提升专业领域识别准确率
- **历史记录** — 自动保存语音输入历史，支持检索和回看

### 🖥️ 桌面体验
- **悬浮球** — 桌面置顶悬浮球，单击呼出/隐藏浮窗
- **紧凑浮窗** — 900×300 精简浮窗，实时转写 + 润色卡片一键发送
- **完整窗口** — 全部功能完整展示，方便管理配置
- **全局热键** — `Alt+Q` 快速切换语音输入
- **系统托盘** — 常驻托盘，后台静默运行
- **文本注入** — 语音识别结果自动输入到当前聚焦的应用

### 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 桌面壳 | Electron 28 + TypeScript | 主进程/渲染进程分离 |
| 前端 | React 18 + Vite | 双模式渲染（浮窗/完整窗口） |
| 后端 | Python FastAPI + Uvicorn | REST API + WebSocket |

## 快速开始

### 环境要求

- Windows 10/11
- Python 3.10+
- Node.js 18+
- FFmpeg（用于音频转码）

### 安装与运行

```bash
# 1. 安装 Python 依赖
cd backend
pip install -r requirements.txt

# 2. 安装 Node 依赖
cd frontend
npm install
cd ../electron
npm install
cd ..

# 3. 启动前端开发服务器
cd frontend && npm run dev &

# 4. 启动 Python 后端
cd backend && python -m uvicorn main:app --reload --port 8000 &

# 5. 启动 Electron 桌面应用
cd electron && npx tsc && npx electron .
```

> 首次启动 FunASR 会自动下载模型（约 1.5GB），请耐心等待。

## 使用说明

1. **启动应用** — 运行后系统托盘出现麦克风图标，桌面显示悬浮球
2. **呼出浮窗** — 按 `Alt+Q` 或点击悬浮球
3. **开始录音** — 点击「开始录音」按钮，对着麦克风说话
4. **实时转写** — 浮窗中实时显示识别文字
5. **场景切换** — 在办公/聊天/创作场景间切换，获取不同优化建议
6. **发送文字** — 点击润色卡片或「发送原文」，文字自动输入到当前应用
