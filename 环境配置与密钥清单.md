# 智能语音输入法 — 环境配置与密钥清单

## 概述

本项目的 AI 架构采用分层设计，**MVP 模式无需任何 API 密钥即可运行**。
以下密钥仅在需要使用大模型能力（办公润色、创作整理增强版）时才需配置。

---

## 快速判断

```
当前是否免费可用？
  是 → 你什么都不需要提供，直接双击 start.bat 即可

是否需要大模型增强？
  是 → 只需配置 1 个密钥（二选一）：
    ├─ Anthropic API Key（推荐，适配更好）
    └─ OpenAI API Key（备选）
```

## 二、可选配置 — 大模型 API

### 2.1 Anthropic API（推荐）

用于 **办公润色增强版** 和 **创作整理增强版**。

| 项目 | 说明 |
|------|------|
| 服务商 | Anthropic |
| 官网 | https://console.anthropic.com/ |
| 所需密钥 | `ANTHROPIC_API_KEY` |
| 默认模型 | `claude-3-sonnet-20241022` |
| 可选模型 | `claude-3-opus-20240229`（更强）、`claude-3-haiku-20240307`（更快更便宜） |
| 用途 | 文本润色、会议纪要整理、内容扩写、提纲生成 |

**配置方式**：

在 `backend/` 目录下创建 `.env` 文件：

```ini
ANTHROPIC_API_KEY=sk-ant-你的密钥
ANTHROPIC_MODEL=claude-3-sonnet-20241022
MVP_MODE=false
LLM_ENABLED=true
```

### 2.2 OpenAI API（备选）

| 项目 | 说明 |
|------|------|
| 服务商 | OpenAI |
| 官网 | https://platform.openai.com/ |
| 所需密钥 | `OPENAI_API_KEY` |
| 默认模型 | `gpt-4o-mini` |
| 可选模型 | `gpt-4o`、`gpt-4-turbo` |
| 用途 | 文本润色、内容整理（与 Claude 二选一） |

**配置方式**：

```ini
OPENAI_API_KEY=sk-你的密钥
OPENAI_MODEL=gpt-4o-mini
MVP_MODE=false
LLM_ENABLED=true
```

## 三、生产环境可选 — ASR 语音识别引擎

MVP 阶段使用浏览器内置的 Web Speech API，以下为生产环境备选方案。

### 3.1 FunASR（推荐，开源免费）

| 项目 | 说明 |
|------|------|
| 类型 | 开源本地部署 |
| 官网 | https://github.com/alibaba-damo-academy/FunASR |
| 安装 | `pip install funasr` |
| 硬件 | CPU 可运行，GPU 效果更佳 |
| 是否需要密钥 | ❌ 无需 |
| 中文效果 | 优秀（优于 Whisper） |
| 配置 | `ASR_ENGINE=funasr` |

### 3.2 Whisper（备选，开源免费）

| 项目 | 说明 |
|------|------|
| 类型 | 开源本地部署 |
| 官网 | https://github.com/openai/whisper |
| 安装 | `pip install openai-whisper` |
| 硬件 | 建议 GPU |
| 是否需要密钥 | ❌ 无需 |
| 多语言 | 优秀（适合海外版本） |
| 配置 | `ASR_ENGINE=whisper` |

### 3.3 商业 ASR API（生产环境）

## 五、完整配置参考文件

在 `backend/.env` 中配置（不存在则自动使用默认值）：

```ini
# ─── 运行模式 ───
MVP_MODE=true              # true=纯规则，false=启用大模型
LLM_ENABLED=false          # 是否启用大模型能力

# ─── 大模型 (二选一) ───
ANTHROPIC_API_KEY=
ANTHROPIC_MODEL=claude-3-sonnet-20241022

OPENAI_API_KEY=
OPENAI_MODEL=gpt-4o-mini

# ─── ASR 语音识别 ───
ASR_ENGINE=webspeech       # webspeech | funasr | whisper
ASR_DEVICE=cpu

# ─── 缓存 ───
CACHE_ENABLED=true
CACHE_TTL_EMOTION=300      # 情绪缓存 5 分钟
CACHE_TTL_OPTIMIZE=3600    # 优化结果缓存 1 小时

# ─── 其他 ───
MAX_HOTWORDS=100
PUNCTUATION_MODEL=rule     # rule | model
```

## 六、常见问题

**Q: 我什么都不配置可以运行吗？**
A: 可以。MVP 模式零依赖，双击 `start.bat` 即可。

**Q: 只有 1 个 API Key 时效果差别大吗？**
A: 配置 Claude 或 GPT 后，办公润色和创作整理的质量会有明显提升，其他功能不受影响。

**Q: 这些 API 大概需要多少费用？**
A: MVP 模式零费用。启用大模型后，日常使用每月约 $1-5 美元（取决于使用量）。建议仅在需要高质量润色时开启。

**Q: FunASR 和 Web Speech API 有什么区别？**
A: Web Speech API 是浏览器内置，免费但精度一般；FunASR 需要本地部署，中文识别效果更好，支持热词注入。
